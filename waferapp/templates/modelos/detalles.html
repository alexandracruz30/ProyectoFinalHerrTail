📃 Instrucciones de Uso
Cómo utilizar el modelo de detección
  1. Prepare su imagen de oblea siguiendo los requisitos de formato.
  2. Cargue la imagen utilizando la opción de subida o arrastrar y soltar.
  3. Haga clic en "Predecir" para obtener el resultado del modelo.
  4. Interprete los resultados según la guía de referencia de clases.
  5. Explore la visualización de zonas críticas utilizando el mapa de calor.
Requisitos de formato
Dimensiones 26x26 píxeles
Canales 3 canales (RGB)
Formato JPG, PNG, TIFF
Tamaño máximo 2 MB
ℹ Nota: Las imágenes de mayor resolución serán redimensionadas automáticamente.
Integración con sistemas existentes
El modelo puede integrarse a sistemas existentes mediante nuestra API REST. Para más información, consulta la documentación API.
